# ISO 690 pro CSL

## Problémy:
(@) Chybí proměnný pro podtitulky. Když mám v bib souboru
    title={Titulek},
    subtitle={podtitulek},
   dostanu *Titulek podtitulek*. Nezdá se, že by to šlo nějak ovlivnit.
   
## Pozitiva
(@) Podporuje bib soubory pro biblatex, jen musí mít příponu bbx.
